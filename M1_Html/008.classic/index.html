Thuộc tính id trong html

    - id trong html nó giống như là số chứng thư của 1 người
    nó sẽ duy nhất trong trang html và không được phép trùng giữa các thẻ
    id phải viết liền và không có dấu , không có ký tự đặc biệt như @ # $ ...
    tốt nhất chỉ nên có các chữ cái và các chữ số
    - class được sử dụng để gom nhóm giữa các thẻ html mà có các điểm chung
    1 thẻ html thì sẽ có 1 hoặc nhiều class khác nhau
    các class thì cần phân tách bằng dấu cách ví dụ : "mienbac top1 thudo"
    quy tắc tên class viết liền và không có dấu , không có ký tự đặc biệt như @ # $ ...
    tốt nhất chỉ nên có các chữ cái và các chữ số
    - mục đích của thuộc tính class và id là để kết hợp với các ngôn ngữ như css và javascript
    để thực hiện các tác vụ trên trang web
    - trong css thì chọn class bằng dấu . ở đầu ví dụ .mienbac
    - trong css để chọn id sẽ viết #id ví dụ #hanoi
