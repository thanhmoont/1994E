Lưới bootstrap thì cấu tạo bởi 12 cột
    Bên ngoài cùng là khung chứa ( class="container" )
    Bên trong khung chứa là hàng ( class="row" )
    Bên trong hàng là cột ( gọi là column )
    Nếu muốn tạo ra các cột bằng nhau thì sẽ dụng các cột có class là col ( class="col" )
